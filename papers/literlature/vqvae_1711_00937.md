# Neural Discrete Representation Learning (VQ-VAE)

- **arXiv**: 1711.00937 (v2, NeurIPS 2017)
- **Authors**: Aaron van den Oord, Oriol Vinyals, Koray Kavukcuoglu (DeepMind)
- **Published**: 2017-11-02
- **PDF**: https://arxiv.org/pdf/1711.00937v2
- **Category**: cs.LG

## 한 줄 요약

이산(discrete) 잠재 표현을 학습하는 VAE 변종. encoder 출력 z_e(x)에 대해 코드북 e ∈ R[K×D]에서 최근접 이웃 lookup으로 z_q(x)를 얻고, straight-through estimator로 encoder까지 gradient를 흘린다. 코드북 업데이트는 본문 식 (3)의 `||sg[z_e] − e||^2` 항 또는 **Appendix A.1의 EMA 업데이트** 둘 중 하나로 가능.

## 본 프로젝트와의 연결 — `src/models/vector_quantizer.py`

`VectorQuantizerEMA`는 **Appendix A.1 (Exponential Moving Averages) 변종**의 충실한 구현이다. 본문 식 (3)의 3항 손실(reconstruction + codebook + commitment) 중 **codebook 항을 EMA로 대체**한 형태로, 손실에는 commitment 항만 남는다. NBEATSx generic stack의 `h_generic`([B, D=64])에 단일 코드북을 거는 1D 변형이며, `src/models/nbeatsx_vq.py`에서 `GenericStackWithVQ.fc4` 출력 직후에 삽입된다.

## 원본 핵심 식

### 본문 식 (1)(2) — Quantization
- `q(z = k|x) = 1` if `k = argmin_j ||z_e(x) − e_j||_2`, else 0
- `z_q(x) = e_k`

### 본문 식 (3) — 표준 손실 (codebook gradient 방식)
```
L = log p(x | z_q(x))
  + ||sg[z_e(x)] − e||_2^2          # codebook loss (e 학습)
  + β · ||z_e(x) − sg[e]||_2^2      # commitment loss (encoder 학습)
```
- `sg[·]`: stop-gradient 연산자
- `β = 0.25` 권장 (0.1–2.0 범위에서 robust)

### Appendix A.1 — EMA 변종 (이 프로젝트가 채택)
- count: `N_i^t = γ · N_i^{t−1} + (1 − γ) · n_i^t`
- weight: `m_i^t = γ · m_i^{t−1} + (1 − γ) · Σ_j z_{i,j}^t`
- codebook: `e_i^t = m_i^t / N_i^t`
- `γ = 0.99` 권장
- 이 변종을 쓸 때 식 (3)의 codebook 항은 제거되고 commitment 항만 남는다.

## `vector_quantizer.py` ↔ 원본 1:1 매핑

| 요소 | 원본 (Appendix A.1) | `VectorQuantizerEMA` (repo) | 일치 여부 |
|---|---|---|---|
| 최근접 이웃 | `argmin_j ||z − e_j||_2` (식 1) | `dist.argmin(dim=1)` (L62–68) | ✅ 동일 (squared L2) |
| Straight-through | `z_q ← z_e + sg[z_q − z_e]` (Sec 3.2) | `z_st = z + (z_q − z).detach()` (L87) | ✅ 동일 |
| EMA decay γ | 0.99 권장 | `decay=0.99` (L34) | ✅ 동일 |
| commitment β | 0.25 권장 | `commitment_beta=0.25` (L33) | ✅ 동일 |
| count EMA | `N_i ← γ·N_i + (1−γ)·n_i` | `ema_count.mul_(decay).add_(count, alpha=1−decay)` (L75) | ✅ 동일 |
| weight EMA | `m_i ← γ·m_i + (1−γ)·Σz_{i,j}` | `ema_weight.mul_(decay).add_(w, ...)` (L77–78) | ✅ 동일 |
| 코드북 정규화 | `e_i = m_i / N_i` | `(N_i + ε)/(ΣN + M·ε) · ΣN` 후 나눔 (L80–84) | ⚠️ **Laplace smoothing 추가** |
| commit loss | `β · ||z_e − sg[e]||^2` | `β · F.mse_loss(z, z_q.detach())` (L86) | ✅ 동일 (MSE는 mean이라 스케일만 다름) |

## 구조적 차이

1. **텐서 차원 — 1D scalar latent**
   - 원본: 이미지 32×32, 비디오 3D, 오디오 1D **공간 격자**로 N개의 latent. 손실은 N개 latent에 대한 평균.
   - 본 repo: `h_generic ∈ R[B, D]`에 latent **1개**만 부여. NBEATSx hidden vector를 하나의 코드로 양자화 — 격자 구조 없음. (`docstring`이 명시: "Single codebook, no per-stack split (MVP scope: h_generic only).")

2. **Laplace smoothing on cluster size** (L80–84)
   - 원본 Appendix A.1: 그대로 `e_i = m_i / N_i`. 한 번도 hit되지 않은 코드는 `N_i → 0`이라 division blow-up.
   - 본 repo: `normalized = (N_i + ε) / (ΣN + M·ε) · ΣN`로 라플라스 평활 후 나눔. 이는 **DeepMind Sonnet 공식 레퍼런스 구현**(`sonnet/src/nets/vqvae.py`)의 트릭이 그대로 정착한 형태로, 원본 논문에는 없지만 모든 모던 VQ-VAE 구현이 채용한다. 결과적으로 "dead code"가 발생해도 수치적으로 안정.

3. **손실 항 구성**
   - 원본 본문(Eq. 3): reconstruction + codebook + commitment 3항.
   - 원본 Appendix A.1 EMA 변종: codebook 항 제거 → reconstruction + commitment 2항.
   - 본 repo: `forward`가 **commitment만** 반환. reconstruction은 NBEATSx 학습 루프(`y_hat ↔ y`)에서 별도 계산되며 STE를 통해 encoder 까지 흐른다 → 의미상 EMA 변종과 동일.

4. **진단(diagnostics) 추가**
   - 원본: `utilization`, `perplexity`, `indices` 출력 없음.
   - 본 repo: codebook collapse 모니터링용으로 세 가지를 반환 (L89–102). 이는 후속 VQ 연구(e.g., Jukebox, RQ-VAE)에서 표준화된 진단치.

5. **Prior 학습 — 본 repo에 없음**
   - 원본: VQ-VAE 학습 후 PixelCNN/WaveNet으로 `p(z)` 학습 → ancestral sampling으로 생성.
   - 본 repo: peak forecasting이라 generation prior가 불필요. discrete latent를 **표현(분석/라우팅) 용도**로만 사용.

6. **차원 매개변수**
   - 원본 실험: ImageNet K=512, D=256; speech K=512.
   - 본 repo 기본: `num_embeddings=32, embedding_dim=64` — UMass 가구 수와 NBEATSx `d_model`에 맞춰 매우 작게 잡힘.

7. **코드북 초기화**
   - 원본: 미명시.
   - 본 repo: `N(0, 0.1)` (L44) — 작은 스케일의 가우시안. encoder hidden과 스케일 정합을 위함으로 보임.

## 본 프로젝트에서 변하지 않은 핵심 invariant

- EMA decay `γ = 0.99`, commitment `β = 0.25` 모두 원본 권장값 그대로.
- STE 정의 (`z + (z_q − z).detach()`)는 원본 Sec 3.2와 비트 단위로 동일.
- Squared L2 nearest-neighbor.

## 사용 맥락

- v05/v07 `Peak-VQ`(post-hoc 1-shot K-means)는 **별도 모듈** `VectorQuantizerKMeans`로, 이 EMA-VQ와 다르다 (CLAUDE.md 명시: iterative federated KMeans는 TAR attack 우려로 보류).
- `VectorQuantizerEMA`는 **end-to-end 학습 가능한 EMA-VQ**가 필요한 경우 (e.g., NBEATSx와 함께 동시 학습) 사용한다. 두 VQ는 학습 시점·집계 방식이 다른 별개의 컴포넌트로 공존.
